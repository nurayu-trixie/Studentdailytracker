
  #  Student Daily Tracker

## Deskripsi

**Student Daily Tracker** adalah aplikasi berbasis web yang dirancang sebagai media pembelajaran bagi siswa SMK untuk memahami konsep dasar pengolahan data, yaitu:

* Input Data
* Proses Data
* Visualisasi Data

Aplikasi ini membantu siswa mencatat nilai harian, memantau progres belajar, serta menampilkan data dalam bentuk statistik dan grafik sederhana.

---

##  Tujuan

Aplikasi ini dibuat untuk:

* Melatih pemahaman konsep **IPO (Input – Process – Output)**
* Membantu siswa memonitor perkembangan nilai
* Mengenalkan pengolahan data sederhana
  

---

##  Fitur Utama

###  Input Data

* Menambahkan nilai harian
* Memilih kategori (Tugas, Ulangan, UAS)
* Menyimpan tanggal otomatis

###  Pengolahan Data

* Menghitung rata-rata nilai
* Menentukan nilai tertinggi
* Menghitung progres terhadap target

###  Visualisasi Data

* Grafik batang (bar chart)
* Tampilan dashboard statistik

###  Target Belajar

* Menentukan target nilai
* Menampilkan persentase progres
* Improvisasi nilai tugas ataupun ulangan

###  Penyimpanan Data

* Menggunakan **Local Storage**
* Data tidak hilang saat browser ditutup

---

##  Konsep yang Dipelajari

* Array & Object di JavaScript
* CRUD (Create, Read, Update, Delete)
* Local Storage
* Perhitungan statistik sederhana
* Visualisasi data
* UI/UX dasar

---

##  Cara Menjalankan

1. Download atau clone repository ini
2. Buka folder project
3. Jalankan file `index.html` di browser
4. atau buka web https://studentdailytracker-production.up.railway.app/
5. Aplikasi siap digunakan 🎉
